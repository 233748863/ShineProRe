# 技能释放逻辑测试说明

## 功能概述
C# 版本的技能释放逻辑已完全实现，包括：
- ✅ 技能冷却时间管理
- ✅ 智能技能选择（按优先级）
- ✅ 自动冷却状态跟踪
- ✅ 详细日志输出

## 测试方法

### 方法 1：直接运行程序
```bash
cd ShineProCS
dotnet run --project ShineProCS
```

点击"启动"按钮后，观察控制台输出：
- 技能释放日志（显示技能名称、优先级、冷却时间）
- 每 5 次循环显示一次所有技能的冷却状态

### 方法 2：修改配置测试
编辑 `config/skills.json`，可以测试：
1. 修改技能优先级（Priority）
2. 修改冷却时间（Cooldown）
3. 启用/禁用技能（Enabled）

示例配置：
```json
{
  "Name": "测试技能",
  "KeyCode": 81,
  "Cooldown": 3.0,
  "Priority": 10,
  "Enabled": true
}
```

## 预期行为
1. 引擎启动后，优先释放优先级最高的可用技能
2. 技能释放后进入冷却，在冷却期间不会被选择
3. 当所有技能都在冷却时，显示"暂无可用技能，等待冷却..."
4. 每 5 次循环显示所有技能的冷却状态
