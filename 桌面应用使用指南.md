# 🎮 游戏自动化脚本 - 桌面应用使用指南

## 📋 快速开始

### 1. 安装依赖库
```bash
# 安装必要的依赖库
pip install keyboard pystray pillow
```

### 2. 启动应用
```bash
# 运行桌面应用启动器
python 桌面应用启动器.py
```

## 🖥️ 界面介绍

### 迷你主界面
- **尺寸**: 300x200 像素，小巧不挡视线
- **位置**: 屏幕右上角，始终置顶
- **功能**: 显示脚本运行状态和关键信息

**界面元素**:
- 🟢🔴 **状态指示器**: 显示脚本运行状态
- 📊 **性能指标**: 响应时间、执行次数等
- ⚙️ **控制按钮**: 启动、暂停、隐藏、退出

### 系统托盘
- **图标**: 红色方块图标
- **右键菜单**: 提供完整控制功能
- **后台运行**: 界面隐藏后仍在后台运行

## ⌨️ 快捷键操作

### 核心快捷键
| 快捷键 | 功能 | 使用场景 |
|--------|------|----------|
| `Ctrl+Shift+P` | 暂停/继续脚本 | 游戏中快速控制 |
| `Ctrl+Shift+H` | 显示/隐藏界面 | 切换界面可见性 |
| `Ctrl+Shift+Q` | 安全退出应用 | 完全退出程序 |

### 鼠标操作
- **界面按钮**: 点击控制按钮进行操作
- **系统托盘**: 右键图标打开菜单
- **窗口关闭**: 点击关闭按钮隐藏到托盘

## 🎮 游戏使用流程

### 标准流程
```
1. 启动游戏
2. 运行脚本 (python 桌面应用启动器.py)
3. 游戏界面显示迷你窗口
4. 点击"启动"按钮开始脚本
5. 点击"隐藏"或按 Ctrl+Shift+H 隐藏界面
6. 游戏中按 Ctrl+Shift+P 控制暂停/继续
7. 游戏结束后按 Ctrl+Shift+Q 安全退出
```

### 推荐配置
1. **游戏设置**: 窗口模式或全屏窗口化
2. **脚本位置**: 将界面放置在屏幕角落
3. **快捷键记忆**: 记住核心快捷键 Ctrl+Shift+P

## ⚙️ 功能详解

### 运行状态管理
- **启动**: 初始化脚本引擎并开始循环
- **暂停**: 临时停止脚本执行（快捷键控制）
- **停止**: 完全停止脚本运行
- **继续**: 从暂停状态恢复执行

### 状态监控
- **实时状态**: 运行中/已停止/暂停中
- **性能指标**: 平均响应时间、执行次数、成功率
- **模式信息**: 当前运行的脚本模式

### 系统集成
- **系统托盘**: 后台运行，不干扰游戏
- **快捷键**: 全局热键，游戏内可用
- **自动恢复**: 异常情况自动恢复

## 🔧 故障排除

### 常见问题

**Q: 快捷键不生效**
A: 检查是否安装keyboard库，尝试以管理员权限运行

**Q: 系统托盘不显示**
A: 检查pystray和PIL库安装，某些系统需要额外配置

**Q: 脚本无法启动**
A: 检查技能循环引擎文件是否存在，查看错误日志

**Q: 界面卡顿**
A: 降低界面刷新频率，或完全隐藏界面使用快捷键

### 性能优化建议
1. **界面隐藏**: 游戏时隐藏界面减少资源占用
2. **关闭日志**: 在性能监控中关闭详细日志输出
3. **调整频率**: 根据游戏需求调整检测频率

## 📊 状态指示说明

### 颜色编码
- 🟢 **绿色**: 脚本正常运行中
- 🔴 **红色**: 脚本已停止
- 🟡 **黄色**: 脚本暂停中
- 🔵 **蓝色**: 脚本初始化中

### 性能指标
- **响应时间**: <0.1s 优秀, 0.1-0.3s 良好, >0.3s 需优化
- **成功率**: >95% 正常, 90-95% 警告, <90% 需要检查
- **执行次数**: 监控脚本运行稳定性
